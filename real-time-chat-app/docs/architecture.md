\# System Architecture



\## Frontend

React + Vite



Responsibilities:

\- Authentication UI

\- Dashboard rendering

\- Room navigation

\- Private messaging

\- Socket event handling

\- Presence indicators



\---



\## Backend

Node.js + Express



Responsibilities:

\- REST API

\- JWT authentication

\- Room management

\- Message persistence

\- User search

\- Socket event orchestration



\---



\## Database

MongoDB



Collections:

\- users

\- rooms

\- messages



\---



\## Real-time Layer

Socket.io



Events:

\- join\_room

\- leave\_room

\- send\_message

\- receive\_message

\- user\_typing

\- stop\_typing

\- read\_receipt

\- user\_online

\- user\_offline
